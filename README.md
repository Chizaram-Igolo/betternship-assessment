## Description of Repo

Included in this repo are the Backend application (Node.js/Express JS) and Frontend application (in React.js) for the CRUD application assessment I was given.

### Fully implemented in Backend application:
- GET
- POST
- DELETE

### Not fully implemented:
- PATCH

### Fully implement in Frontend Application:

- GET - Lists display of Payments
- POST - Provides form to add payment to BE data store
- DELETE - Provides button to remove payment

### Not fully implemented:
- PATCH
